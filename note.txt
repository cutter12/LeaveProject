1. User เข้าระบบ Login (สิทธิแบบ User ทั่วไป)
2. กรอกข้อมูลการลา
3. ข้อมูลการลา รออนุมัติ (เก็บไว้ในฐานข้อมูล) ** เมื่ออนุมัติแล้วจึงนำข้อมูลตัวนี้ไปเก็บไว้ใน PIS การลา เคลียข้อมูลทุกๆ เดือน
4. ส่งอีเมลไปแจ้ง manager มีเนื้อหาแจ้งเหตุผลในการลา สถานะต่างๆ จำนวนวันลา *** และแจ้ง manager ให้เข้าระบบด้วย user ตัวเองเพื่อเข้าไปตรวจสอบและอนุมัติ

5. Manager เข้าสู่ระบบ (สิทธิแบบ Manager) มีข้อมูลการลาที่รออนุมัติ manager ติ๊กถูกเพื่ออนุมัติ

6. ข้อมูลบันทึกลง PIS การลา
